本技能旨在引導建立獨具特色、達生產級水準的前端介面，避免產生常見的「AI 塑膠感 (AI slop)」美學。請實作真正可運作的程式碼，並對美學細節與創意選擇投入極高的關注。

使用者會提供前端需求：要建構的元件、頁面、應用程式或介面。他們可能會提供有關目的、目標受眾或技術限制的上下文。

## 設計思維

在開始編寫程式碼之前，請先理解上下文，並確立一個大膽（BOLD）的美學方向：

- **目的**：這個介面解決了什麼問題？誰會使用它？
- **調性**：選擇一種極致風格：極簡冷酷、極繁混亂、復古未來派、有機/自然、奢華/精緻、俏皮/玩具感、社論/雜誌風、粗獷主義/原始、裝飾藝術/幾何、柔和/粉彩、工業/實用等。有非常多種風格可供選擇。將這些作為靈感，但請設計出忠於該美學方向的作品。
- **限制**：技術需求（框架、效能、無障礙設計）。
- **差異化**：是什麼讓這個設計令人難忘（UNFORGETTABLE）？有什麼是會讓人記住的亮點？

**極度重要**：選擇一個明確的概念方向並精準執行。大膽的極繁主義和精緻的極簡主義都行得通——關鍵在於「刻意為之（intentionality）」，而非單純追求強烈。

接著，實作可運作的程式碼（HTML/CSS/JS、React、Vue 等），且必須符合以下條件：

- 達生產級水準且功能完善
- 視覺效果驚豔且令人難忘
- 具備高度凝聚力且美學觀點明確
- 每個細節都經過精心打磨

## 前端美學指南

專注於以下幾點：

- **字體排印（Typography）**：選擇美觀、獨特且有趣的字體。避免使用 Arial 和 Inter 等大眾化字體；改用能提升前端美學的獨特選擇；選擇出乎意料、充滿個性的字體。將獨特的展示型字體（display font）與精緻的內文字體（body font）搭配使用。
- **色彩與主題**：致力於打造具凝聚力的美學。使用 CSS 變數（variables）來保持一致性。帶有強烈點綴色的主導色彩，其效果遠勝於保守、平均分配的調色盤。
- **動態效果（Motion）**：使用動畫來呈現視覺效果與微互動。在純 HTML 中優先使用純 CSS 解決方案。若使用 React，請盡量使用 Motion 函式庫。將重點放在具高衝擊力的瞬間：一次精心編排、帶有交錯揭示效果（使用 animation-delay）的頁面載入，會比零散的微互動帶來更多驚喜。善用能帶來驚喜的滾動觸發（scroll-triggering）與懸停狀態（hover states）。
- **空間構圖（Spatial Composition）**：出乎意料的版面配置。不對稱設計。元素重疊。對角線視覺流動。打破網格的元素。大方的留白（負空間）或是受控的高密度排版。
- **背景與視覺細節**：創造氛圍與層次感，而不是預設使用單色背景。加入與整體美學相符的情境效果與材質紋理。運用具創意的形式，例如：漸層網格、雜訊紋理、幾何圖案、多層次透明度、戲劇化的陰影、裝飾性邊框、自訂游標以及顆粒疊加效果（grain overlays）。

**絕對不要**使用千篇一律的 AI 生成美學，例如過度使用的字體（Inter、Roboto、Arial、系統預設字體）、老套的配色方案（特別是白底配紫色漸層）、可預測的排版與元件模式，以及缺乏情境專屬特色的模版化設計。

發揮創意進行詮釋，並做出讓人感覺是「真正為該情境量身打造」的出乎意料之選。不應該有任何兩個設計是相同的。在亮色與暗色主題、不同的字體、不同的美學之間變化。**絕對不要**在多次生成中都侷限於相同的常見選擇（例如一直使用 Space Grotesk）。

**重要**：實作的複雜度必須與美學願景相符。極繁主義的設計需要精巧的程式碼，並搭配大量的動畫與特效。極簡或精緻的設計則需要克制、精準，並仔細關注間距、字體排印與微妙的細節。優雅來自於將願景執行得盡善盡美。

請記住：AI（Claude）具備創造非凡作品的能力。不要自我設限，展現出當你跳脫框架思考，並完全投入於獨特願景時，真正能創造出什麼樣的成果。
